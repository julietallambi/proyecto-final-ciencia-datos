# proyecto-final-ciencia-datos
Nuestro proyecto se basa en un análisis del costo de la educación internacional. 
En este repositorio se encuentran los datos utilizados ("International_Education_Costs.csv"), el documento con el análisis de los mismos en .qmd y .pdf ("Proyecto Final CDR.qmd"y ) y el código fuente de la aplicación shiny ("app.R").
